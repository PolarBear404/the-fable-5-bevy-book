# 前言

> **重点提醒**
>
> 本教程完全由 **Claude Fable 5** 生成。请把它视为一份 AI 生成的 Bevy 中文学习教程，而不是 Bevy 官方文档或官方授权译本。

Bevy 是一个用 Rust 编写的免费开源游戏引擎，以 ECS（Entity Component System）架构为核心，
内置渲染、UI、资产、音频、输入、动画等游戏开发所需的完整能力。

本书的目标，是像 *The Rust Book* 之于 Rust 那样，为 Bevy 提供一份系统、循序渐进、
可以从头读到尾的中文教程。

## 本书假设你

- 已掌握 Rust 基础：所有权与借用、trait、泛型、闭包、模块系统。本书不解释 Rust 语言本身。
- 没有任何游戏引擎或游戏开发经验。ECS、渲染管线、资产、坐标变换这些概念都会从零讲起。

## 版本与代码

全书基于 **Bevy 0.19.0** 编写。书中出现的每一段代码都来自本书仓库 `code/` 目录下
真实存在、通过编译验证的项目，而不是手写在文中的片段。
