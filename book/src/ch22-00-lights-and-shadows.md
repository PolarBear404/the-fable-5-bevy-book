# 光照与阴影

得月楼开张第二天，老雷就收到一沓看客的条子。夸布景立体的有，挑刺的更多：“大白天演夜戏”“影子呢？桌子怎么浮在台上”“那颗鎏金绣球黑得像口锅”。老鲁委屈——布景是他一刀一凿铸的，没毛病。毛病出在光上：整座园子从开工到现在，只点过一盏没拆封的堂灯，位置没挪过、亮度没调过、影子压根没开过。老雷托人从府城请来一位灯光师傅，姓烛，人称**老烛**——干这行的规矩是灯不离手、表不离身：一只皮箱里全是灯，怀里揣一块测光表。

老烛围着园子转了一圈，只说了一句行话：“光是半台戏。”于是开了本章的工，把光这半台戏一件件请上台：

- **拆堂灯**——`PointLight` 的旋钮逐个拧开：流明、光程、发光体大小；顺手把引擎兜底的环境光总闸拉了，看看一盏灯的真本事；
- **测光表**——同一盏灯，四档曝光四种画面：`Exposure` 是相机吃光的口径，灯的数值只是故事的一半；
- **追光**——`SpotLight` 盯着角儿走，内沿外沿拧出软边硬边；
- **请太阳**——`DirectionalLight` 只有方向没有位置，照度以勒克斯与现实对表；亲手把太阳搬走二十米，看画面动不动；
- **灯箱**——第四种灯 `RectLight`：一整块发光的面，还要跨过一道 cargo feature 的门槛；
- **影子开箱**——`shadow_maps_enabled` 一个开关，三种灯三种影子；影子贴图的原理、分辨率的预算、级联的铺法；
- **影子的毛病**——粉刺与彼得潘，偏置的两难；接触阴影补根须，`NotShadowCaster` 让绸幕隐身；
- **环境光照**——没有灯也能亮：环境光的平光兜底，环境贴图让镜面球终于照见世界；
- **星空天幕**——一条竖布裁成六面天：`Skybox` 画天，运行时滤波让天发光；顺带亲眼看一次天幕尺寸不对的当场翻车；
- **真的天**——`Atmosphere` 大气散射：太阳一低，天自己烧红，不用你调一个颜色；
- **镜厅**——反射探针划地盘，视差校正让反射贴着墙走，两间厅的天光无缝过渡；
- **晨雾**——体积雾三件套，柱缝里漏下来的光柱；
- **昼夜切换台**——全章合龙：正午、黄昏、夜戏、晨雾，一个数字键一套光。

这些类型分住几个 crate：四种光源、环境光、探针、天幕、大气与体积雾的**数据组件**全在 `bevy_light`；把它们画出来的渲染端在 `bevy_pbr`（连同相机侧的 `AtmosphereSettings`、`ContactShadows`）；曝光 `Exposure` 在 `bevy_camera`。日常不用记门牌——灯的全家、环境光、探针都在 `bevy::prelude`，其余从 `bevy::light`、`bevy::pbr`、`bevy::camera` 显式引入，用到时正文会点名。

配套 crate 是 `code/ch22-lighting`。它的 `Cargo.toml` 比往章多一样东西——两个 cargo feature：

```toml
{{#include ../../code/ch22-lighting/Cargo.toml:deps}}
```

`area_light_luts` 是 22.5 节灯箱的入场券，`debug` 让报错里印出系统名（第 4 章起的老习惯）。本章的图片资产（星空天幕、镜厅贴图）由 `scripts/make_ch22_assets.py` 一键合成：

```console
py -3.11 scripts/make_ch22_assets.py
```

开工。
