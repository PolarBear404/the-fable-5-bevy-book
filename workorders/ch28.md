## 源码侦察备忘（vendor/bevy@0.19.0 已核）

- `Node`（ui_node.rs:492）：border_radius 是 **Node 字段**（BorderRadius 类型非组件）；required:
  ComputedNode/ComputedStackIndex/ContentSize/ComputedUiTargetCamera/ComputedUiRenderTargetInfo/
  UiTransform/BackgroundColor/BorderColor/FocusPolicy/ScrollPosition/Visibility/ZIndex。
- `BorderColor` 分四边 {top,right,bottom,left}，`BorderColor::all()` 助手；BackgroundColor 默认
  Color::NONE 透明。
- `Val`（geometry.rs）：Auto/Px/Percent/Vw/Vh/VMin/VMax；便捷函数 `px()/percent()/vw()/vh()/
  vmin()/vmax()/auto()`（官方示例主用此风格）；Percent 参照：margin/padding/border 一律参照
  **父宽**，left/right 参照父宽、top/bottom 参照父高，width/height 各自参照对应轴。
- `UiScale`（lib.rs:120）只乘 Val::Px（注释「only affect fixed ui values like Val::Px」）。
- 布局系统 `ui_layout_system` 在 **PostUpdate**（UiSystems::Layout）→ 首帧 Startup 里读
  ComputedNode 全零；ComputedNode 存**物理像素**，`inverse_scale_factor` 换算逻辑像素。
- 无相机：`propagate_ui_target_cameras`（update.rs:125）拿不到相机就 Entity::PLACEHOLDER，
  **无任何 warn 路径**——UI 静默不渲染（哑巴坑，待实测）。
- 默认 UI 相机（DefaultUiCamera::get，ui_node.rs:2990）：单个 IsDefaultUiCamera 优先；
  多个 IsDefaultUiCamera → once! warn「Two or more Entities with IsDefaultUiCamera found…」；
  否则取渲染到主窗口的相机里 **order 最大**者（并列取实体号大者）。
- `GridPlacement::span(0)` 等 → `expect("Invalid span value of 0.")` 运行期 panic（ui_node.rs:2077）。
- `ZIndex(i32)` 只在**兄弟间**比较；`GlobalZIndex(i32)` 使节点脱离树序全局排（ui_stack_system：
  根节点+带 GlobalZIndex 者一起按 (global, local) 排序成 UiStack，back-to-front）。
- `Display::None` 不布局不渲染；`Visibility::Hidden` 不渲染但**占位**（Node 文档明说）。
- `bevy_ui_debug` feature（根 Cargo.toml:277）→ `GlobalUiDebugOptions` 资源 + 每节点
  `UiDebugOptions` 组件覆盖；三框 outline_border_box（默认 true）/padding_box/content_box、
  line_width、line_color_override、show_hidden、show_clipped、ignore_border_radius。
- `ImageNode`：字段 color/image/texture_atlas/flip_x/flip_y/rect/image_mode/visual_box；
  `NodeImageMode::{Auto,Stretch,Sliced(TextureSlicer),Tiled{tile_x,tile_y,stretch_value}}`；
  Auto 用图的原始尺寸参与测量；默认 image=透明 1×1。
- UI `Text`（widget/text.rs）：require Node/TextLayout/TextFont/TextColor/LineHeight/LetterSpacing/
  TextNodeFlags/ContentSize/**FontHinting::Enabled**（UI 文本默认开 hinting，Text2d 不开——可提）；
  TextShadow{offset,color} 默认偏移 (4,4)。
- 渐变：`BackgroundGradient(Vec<Gradient>)`/`BorderGradient`，Gradient::{Linear,Radial,Conic}。
- 官方示例主参考：ui/layout/{flex_layout,grid,z_index,display_and_visibility,size_constraints}.rs、
  ui/ui_target_camera.rs、ui/ui_scaling.rs、ui/images/*.rs、ui/styling/*.rs。

## 坑与报错素材

- ① 无相机：UI 静默不渲染、零警告（源码已核无 warn 路径；待实测画面与日志）。
- ② `GridPlacement::span(0)`：运行期 panic「Invalid span value of 0.」（待逐字实录）。
- ③ grid_* 字段配在 Flex 容器：静默无效（taffy 只在 Display::Grid 读 grid 字段；待实测）。
- ④ 首帧 ComputedNode 全零：布局在 PostUpdate，Startup 帧读不到（待实测输出）。
- ⑤ 多个 IsDefaultUiCamera：once! warn（待实测逐字）。
- feature 门 `bevy_ui_debug`：不开则 GlobalUiDebugOptions 不存在（E0432/E0433，脚注级）。
- UiScale 只乘 Px（待实测对比）。
- ZIndex 只管兄弟之间——跨树比较无效，须 GlobalZIndex（待实测）。
