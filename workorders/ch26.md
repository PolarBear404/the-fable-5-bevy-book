## API 核对台账（vendor/bevy v0.19.0 已确认）

| API | 位置 | 关键事实 |
|---|---|---|
| `Hdr` | bevy_camera::components:89 | 单元结构 marker 组件；「中间纹理 HDR」，不是显示器 HDR 输出 |
| `Tonemapping` | bevy_core_pipeline::tonemapping:119 | 9 变体；默认 TonyMcMapface；组件在相机上；Camera3d 经 register_required_components 自动带（core_3d/mod.rs:108） |
| `DebandDither` | 同上:383 | enum Disabled/Enabled；Camera3d 默认 Enabled（core_3d/mod.rs:104 register_required_components_with） |
| `Exposure` | bevy_camera::camera:232 | ev100 f32；预设 SUNLIGHT=15/OVERCAST=12/INDOOR=7/BLENDER=9.7（默认 BLENDER） |
| `PhysicalCameraParameters` | bevy_camera::camera:288 | aperture_f_stops/shutter_speed_s/sensitivity_iso/sensor_height |
| `Bloom` | bevy_post_process::bloom::settings | `#[require(Hdr)]`；intensity 默认 0.15；预设 NATURAL/ANAMORPHIC/OLD_SCHOOL/SCREEN_BLUR；prefilter{threshold,threshold_softness}；composite_mode EnergyConserving/Additive；scale（anamorphic 用）；max_mip_dimension 512；extract 过滤 With<Hdr> |
| `DepthOfField` | bevy_post_process::dof:77 | mode/focal_distance(默认10)/sensor_height(0.01866)/aperture_f_stops/max_circle_of_confusion_diameter(64)/max_depth(INF)；`from_physical_camera`；WebGL2 直接停摆（DEPTH_PREPASS_TEXTURE_SUPPORTED=false，info 日志一次） |
| `DepthOfFieldMode` | 同上:121 | Bokeh / Gaussian（默认 Gaussian） |
| `MotionBlur` | bevy_post_process::motion_blur:76 | `#[require(MotionVectorPrepass)]`；shutter_angle 默认 0.5；samples 默认 1（3 采样）；samples=0 等于关 |
| `Vignette` | effect_stack::vignette:30 | intensity 1.0/radius 0.75/smoothness 5.0/roundness 1.0/center(0.5,0.5)/edge_compensation 1.0/color BLACK；intensity ≤1e-4 时 extract=None（零开销关闭） |
| `LensDistortion` | effect_stack::lens_distortion:23 | intensity 默认 0.5（正=桶形、负=枕形）/scale/multiplier/center/edge_curvature（k2 间接控制，推荐 0） |
| `ChromaticAberration` | effect_stack::chromatic_aberration:45 | color_lut（None=红绿蓝 3×1）/intensity 0.02/max_samples 8；Inside GDC2016 实现 |
| `AutoExposure` | bevy_post_process::auto_exposure::settings:30 | `#[require(Hdr)]`；range -8..=8 EV/filter 0.1..=0.9/speed_brighten 3.0?/speed_darken/exponential_transition_distance/metering_mask/compensation_curve；**需要 compute shader，WebGL2 不可用**（doc 原文）；需要 AutoExposurePlugin？（非默认插件——待查 plugin 注册方式） |
| `Msaa` | bevy_render::view:240 | **组件**（不是资源！）；enum Off=1/Sample2/Sample4(默认)/Sample8；Camera 的 required component（bevy_render camera.rs:65）；web 只支持 1 或 4 |
| `Fxaa` | bevy_anti_alias::fxaa:57 | enabled/edge_threshold(默认 High)/edge_threshold_min；Sensitivity 五档 Low..Extreme |
| `Smaa` | bevy_anti_alias::smaa:94 | preset：Low/Medium/High(默认)/Ultra |
| `TemporalAntiAliasing` | bevy_anti_alias::taa:115 | `#[require(TemporalJitter, MipBias, DepthPrepass, MotionVectorPrepass)]`；字段只有 reset；**要求 Msaa::Off，否则渲染系统每帧 warn**（taa/mod.rs:152） |
| `ContrastAdaptiveSharpening` | bevy_anti_alias::contrast_adaptive_sharpening:39 | enabled/sharpening_strength 0.6/denoise(别开)；strength clamp 0..1 |
| `SolariPlugins` | bevy_solari::lib | 非默认 feature `bevy_solari`；需 WgpuFeatures::EXPERIMENTAL_RAY_QUERY 等；SolariLighting 组件 + RaytracingMesh3d；另有验证用 pathtracer |
| feature 门 | vendor/bevy/Cargo.toml | bevy_post_process/bevy_anti_alias/tonemapping_luts/smaa_luts 均在默认 3d profile 内；bevy_solari/dlss 非默认 |
| `check_msaa` | core_3d/mod.rs:749 | DeferredPrepass + MSAA≠Off → warn 一次/帧? 并强制 Off（deferred 本章不讲，正文顺带一句即可，不做实验） |

## 示例设计

**主题**：《定妆照》——夜戏开演前，班主老雷请来「盛记照相馆」的盛师傅给戏台拍宣传定妆照。
后处理全是真实摄影/镜头概念（曝光、景深、快门角、暗角、畸变、色差），摄影师傅调相机 = 逐项拨画质开关。
延续戏班宇宙：老雷（班主）、盛师傅（掌机/调参）、场记（打印输出）。道具复用概念：灯笼（emissive 辉光）、
走马灯（旋转→运动模糊）、栏杆/旗杆（细几何→锯齿）。

**场景底子**（各 listing 共用 scene.rs 或 setup 函数）：夜色戏台——
- 台面 + 后墙（中性材质收光）
- 一排灯笼（emissive 强弱不同 → bloom 素材）
- 前/中/后三只道具（琉璃盏近、锣中、旗远 → dof 对焦三档）
- 一盏旋转走马灯（恒速自转 → motion blur）
- 细栏杆条 + 斜旗杆（高对比细几何 → AA 对比）
- 相机固定机位；照明克制（夜景，给 bloom/曝光留动态范围）

**范围修正（2026-07-06 定稿）**：ch22.2 已完整讲过 Exposure（四档、from_physical_camera 注记、
EV 越大越暗 + Figure 22-2）；ch24.3 已讲 emissive_exposure_weight。**本章不再设 Exposure 单节**，
只在 DoF（from_physical_camera 同参联动）与 AutoExposure（自动拨 ch22 那只手动表）处回访。
**必须兑现的前文钩子**：ch15-05（>1 颜色 + Bloom 之门）、ch24-03（bloom 晕光圈）、
ch22-07（接触阴影的噪点指望 TAA 抹平——TAA 节做回访实验）、ch25-14 结尾预告清单。

**Listing 规划（定稿编号）**：

## 实测翻车与修正记录（正文素材）

1. **场太黑病**（多listing 初版）：夜景在默认 EV 9.7 下欠曝→ 修正=相机挂 `Exposure::INDOOR`（ch22.2 回访），
   而不是无脑加灯——正文 26.2 开场可用这个衔接。
2. **景深隐形**（26-05 初版）：默认 45° FOV=短焦广角，f/1.4 在 5m 对焦下 16m 远物弥散圆才 2-3px。
   修正=fov 0.35（53mm 长焦）。**「景深大小随镜头焦距平方走」→ 正文微实验/警示框素材**。
3. **OLD_SCHOOL 门槛失效**（26-03 初版）：珠灯 5.0→1.0 线性递减全在 threshold 0.6 之上→改 8.0×0.5^i 对半递减。
4. **运行时拆装 ContactShadows 崩溃**（26-11 初版）：remove 后再 insert → wgpu Validation Error
   （mesh_view_bind_group 带 contact_shadows、pipeline 没跟上）→ `Quitting the application due to Validation RenderError`。
   疑似 0.19 引擎缓存 bug；listing 改为只拨 Msaa 档（不拆装组件），崩溃路径不入正文，记档于此。
