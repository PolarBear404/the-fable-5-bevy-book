## 1　范围与衔接

**大纲条目（OUTLINE ch24）**：bevy_pbr：StandardMaterial 全参数（金属度/粗糙度/法线/自发光/
透明/clearcoat…以源码为准）、双面、深度偏移；示例：材质球画廊。

**0.19 API 底账（已逐项核对 vendor/bevy，主源 bevy_pbr/src/pbr_material.rs 1-1563）**：

- `StandardMaterial` 字段全景（按讲授分组；默认值出自 `impl Default`，pbr_material.rs:851）：
  - 已讲（ch21.3）：`base_color`（白）、`base_color_texture`（乘法）、`metallic`（0.0）、
    `perceptual_roughness`（0.5，着色器钳 [0.089,1]）
  - 反光度：`reflectance`（0.5 → 4% 反射率，别名 specular_intensity）、`specular_tint`（白；
    **文档自相矛盾**："modulate for non-metals" vs "no effect for non-metals"——uniform 里
    `reflectance = specular_tint.rgb * reflectance`（:1190），须实测金属/非金属谁的高光变色；
    deferred 路径不支持）
  - 自发光：`emissive`（LinearRgba::BLACK；单位 cd/m²=nit，通道值可 >1；**不照亮邻居**）、
    `emissive_texture`（相乘）、`emissive_exposure_weight`（0.0=不受曝光影响，1.0=全受）
  - 贴图化：`metallic_roughness_texture`（**B=金属度、G=粗糙度**，与标量相乘——默认标量
    metallic=0 时贴图无效，坑）、`occlusion_texture`（R=AO，ORM 打包惯例、glTF 同款）
  - 法线：`normal_map_texture`（需要 UV+法线+**切线**；图元不带切线→**静默平面**，管线只在
    layout 有切线才推属性，render/mesh.rs:3319）、`flip_normal_map_y`（DirectX 风格 y 翻转）、
    线性色空间规矩 is_srgb=false（字段文档自带 load_builder 示例）；
    `Mesh::generate_tangents`/`with_generated_tangents`（mikktspace，要求 TriangleList+
    POSITION+NORMAL+UV_0，Result；**bevy_mikktspace 在 3d_api 默认集合里**，vendor Cargo.toml:231）
  - 视差：`depth_map`（**白=深、黑=凸**）、`parallax_depth_scale`（0.1，相对贴图占的世界尺寸）、
    `parallax_mapping_method`（Occlusion 默认 / Relief{max_steps}，DEFAULT_RELIEF_MAPPING=5 步）、
    `max_parallax_layer_count`（16.0，锯齿→调大）
  - 清漆：`clearcoat`（0.0=无）、`clearcoat_perceptual_roughness`（0.5）；三个贴图字段锁在
    `pbr_multi_layer_material_textures` feature（非默认）
  - 各向异性：`anisotropy_strength`（0..1，拉长高光）、`anisotropy_rotation`（弧度，切线系里转），
    **也吃切线**；贴图锁 `pbr_anisotropy_texture`（非默认）
  - 透明：`alpha_mode`——AlphaMode 七款（bevy_material/src/alpha.rs）：Opaque/Mask(f32)/Blend/
    Premultiplied/AlphaToCoverage（要 MSAA，否则=Mask(0.5)）/Add（黑=无效果）/Multiply（白=无效果）；
    `From<Color>` alpha<1 自动转 Blend（:943）——**struct 字面量绕过这条**，坑
  - 透光：`diffuse_transmission`（翻转朗伯瓣，纸/叶/瓷，便宜；>0.5 背光面比受光面亮）、
    `specular_transmission`（屏幕空间折射，贵；**Blend 等透明物在它后面画→透过玻璃看不见**）、
    `thickness`（0=薄膜）、`ior`（1.5；表：水1.33/玻璃1.52/钻石2.42）、`attenuation_distance`
    （INFINITY）、`attenuation_color`（要 distance+thickness+transmission 齐活才生效）；
    相机组件 `ScreenSpaceTransmission { steps: 1, quality: Medium }`（transmission/mod.rs:67，
    Camera3d 的 required component；steps=0 退化为只折环境图）；`TransmittedShadowReceiver`
    （bevy_light）；透光贴图三件锁 `pbr_transmission_textures`（非默认）
  - 开关：`double_sided`（背面翻法线，**不管剔除**）、`cull_mode`（Some(Back) 默认，None=双面可见；
    逆时针=正面）、`unlit`（只出 base_color；文档列了一串被忽略项含 alpha_mode——**待实测**）、
    `fog_enabled`（true；配 DistanceFog 相机组件，bevy_pbr/src/fog.rs:55，FogFalloff 四款）、
    `depth_bias`（f32→i32 进管线 key 高 32 位；正=拉近；治 z-fighting）
  - UV：`uv_transform`（Affine2，常量 FLIP_HORIZONTAL/FLIP_VERTICAL + `flip()/flipped()` 方法）；
    每张贴图配 `*_channel: UvChannel`（Uv0/Uv1，用 ATTRIBUTE_UV_1）
  - 只点名进清单：`lightmap_exposure`（配 Lightmap 组件，烘焙光照贴图）、`opaque_render_method`
    （Auto/Forward/Deferred；**diffuse_transmission>0 时 Auto 强制走 Forward**，:1396）、
    `deferred_lighting_pass_id`（→ch37）、`specular_texture`/`specular_tint_texture`
    （锁 pbr_specular_textures）
- 环境地基（ch22 已教不重讲）：Skybox + GeneratedEnvironmentMapLight（PNG 竖条 reinterpret 成
  cubemap 的 ch22.9 手法；**边长必须 2 的幂**，ch22.10 的 panic 教训）；Exposure（ch22.2）
- 官方示例第一参考：3d/pbr.rs（金属度×粗糙度阵列）、blend_modes.rs（五球一字排开+键控）、
  transmission.rs、clearcoat.rs、anisotropy.rs、parallax_mapping.rs、specular_tint.rs、
  texture.rs（alpha 三态）

**24-02 反光度与高光染色**（实跑 ✓）：
```text
小棠：前排白瓷五连——反光度从 0 拧到 1，高光从无到扎眼。
小棠：后排黑釉一对、银器一对，每对右手那颗抹了石绿——猜猜谁认这罐染料？
```
实测结论：①reflectance=0 完全无高光（哑光相），0.25→1.0 逐档增强但**前半段差异细**
（F0=0.16·r² 平方刻度，正文要点破：0/1/4/9/16%）；②specular_tint 石绿：黑釉（非金属）
整颗变墨绿玻璃球，**银（金属）双胞胎逐像素相同、纹丝不动**——文档"no effect for
non-metals"确系笔误（官方 specular_tint.rs 注释与 Filament 规范佐证：金属忽略
reflectance/tint，其反射色来自 base_color）。黑底才能让高光色显形（白底漫反射会淹掉
16% 镜面）——官方示例同款手法（base_color BLACK + reflectance 1.0）。

**24-03 自发光暗房**（实跑 ✓，E/L 键实测）：
```text
小棠：暗房验灯——灯箱一块、橙球三档：1、10、100 尼特。
老烛：素坯挨着 100 尼特坐，一丝光都借不着才对。E 拨曝光，L 换真灯。
老烛：曝光拨到 EV100 = 12——看好两颗青球谁掉了亮。
老烛：曝光拨到 EV100 = 9.7——看好两颗青球谁掉了亮。
老烛：同一个位置换一盏真灯——地板、素坯、邻居全亮了。
```
实测：①亮度阶梯 1/10/100 尼特三档肉眼可辨（10/100/1000 时后两档全被色调映射冲白，
改档；100 尼特已近白——高亮度去饱和是 tonemapping 的正常相）；②素坯球在 100 尼特邻居
旁**完全隐形**（纯黑）；L 键同位置换真点光（60_000 lm 暖橙）→ 地板洇出光池、素坯现出
半边、青球沾橙边——「自发光不是灯」实证；③E 键 EV100 9.7↔12：全场（灯箱、阶梯、
weight=0 青球）纹丝不动，**唯 weight=1.0 青球独自变暗**；且默认曝光下两颗同 80 尼特
青球亮度就不同（weight=0 亮得多）——default 0.0 = 完全不过曝光的秤，写多少是多少。

**24-04 ORM 铜锣**（实跑 ✓，SPACE 三阶段）：
```text
小棠：旧铜锣坯来了——底色一张、ORM 一张，全贴上了。空格换阶段。
老雷：说好的鎏金呢？这锣怎么一身塑料相？
小棠：阶段甲——标量全默认。金属度 0 乘什么都是 0，贴图白贴。
小棠：阶段乙——标量拨到 1，贴图接管：铜是铜，锈是锈。
小棠：阶段丙——同一张图兼任 AO，锈坑里的天光被按了下去。
```
实测：①阶段甲「贴了没反应」坑成立——整球塑料玩具相，零金属；②阶段乙铜区变真金属
（映柔光箱）、锈区哑糙；③阶段丙 AO（R 通道锈=50/255）**只压环境光份额**：受主灯直射的
锈斑几乎不变、背光面/靠环境光的锈斑明显发黑（对比裁片 max diff 53/255）——AO 不减
直接光，正文按此讲。ORM 用 load_builder is_srgb=false（线性数据规矩）。

**24-07 视差**（实跑 ✓，[/]/M/N 键实测）：
```text

小棠：刻痕加深——parallax_depth_scale = 0.12 / 0.16

```
实测：①左（仅法线）图案是「画上去的」、轮廓平；右（+depth_map 0.08）图案凹陷入盖、
边缘有遮挡感；②0.16 + 默认 16 层：螺旋侧壁明显**分层条带**伪影；N→64 层平滑；
M→Relief{5} 同样救；③机位教训：~7° 掠射角下全是伪影且读不出图案，**~20° 斜视角**
（h+1.05/r3.2）最佳；垂直俯视无偏移可言。深度图约定白=深黑=凸（与直觉反），正文点破。

**24-08 清漆**（实跑 ✓）：三球（素身糙红 rough0.85 / +clearcoat 1.0 ccr0.08 /
+clearcoat 1.0 ccr0.45）：素身哑肉；亮清漆=糙红底上浮一层锐白高光+映窗格（剔红漆器相）；
哑清漆=柔光泽。台词见代码。**clearcoat 高光是白的**（涂层是电介质膜，不沾底色）。

**24-09 各向异性 + feature 坑**（实跑 ✓，两态同码同机位）：
- **坑（--no-default-features）**：anisotropy_strength 0.1/0.25/0.4/0.6/1.0 任何非零值
  → 受光半球整片炸白，且各档一模一样（二值坏，非物理变窄）；平行光/点光/带不带环境光
  皆同。零警告零报错。
- **根因（源码实证）**：pbr_fragment.wgsl:766——整段各向异性输入装配（读 strength/
  rotation、算 anisotropy_T/B）锁在 `#ifdef PBR_ANISOTROPY_TEXTURE_SUPPORTED` 内，该
  def 仅在 cargo feature `pbr_anisotropy_texture` 开启时注入（render/mesh.rs:3335）；
  而 StandardMaterialKey::ANISOTROPY 只看 strength>0（pbr_material.rs:1288）→ 灯光
  函数拿未初始化的 Ta/Ba/anisotropy 算 → 炸白。官方 anisotropy.rs 的
  required-features=["jpeg","pbr_anisotropy_texture"] 不是摆设。
- **修法**：crate 转发 feature（ch16 手法）：`[features] default=["aniso"]
  aniso=["bevy/pbr_anisotropy_texture"]`；默认全绿，实验用 --no-default-features 看坏。
- **好态**：0 圆高光 / 0.6 高光拉环+纵向光带 / 1.0 亮带（纬线环）/ 1.0+rotation π/2
  亮脊转 90°（经线向）——拉丝铜锅底质感，球须先 with_generated_tangents。
- 开 feature 全量重编 bevy_pbr 约 1m41s（正文一句话预告）。

**24-10 alpha 七款**（实跑 ✓）：
```text
小棠：1 号纱幕——alpha_mode::Opaque。
小棠：2 号纱幕——alpha_mode::Mask(0.5)。
小棠：3 号纱幕——alpha_mode::Blend。
小棠：4 号纱幕——alpha_mode::AlphaToCoverage。
小棠：前排左幽灵右茶镜；居中一对「同款半透明」——右边那颗怎么是实心的？
```
实测：①Opaque 纱幕=黑底整块（alpha 全当 1）；Mask 硬剪影；Blend 软边；A2C 类 Mask 带
抗锯齿边（默认 Msaa 4x 生效）；②Add 幽灵球（unlit+Add）半透发蓝、Multiply 茶镜后面的
东西全变琥珀暗调；③**From<Color> 坑实证**：materials.add(Color::srgba α=0.35) 自动转
Blend=幽灵球，struct 字面量同色=实心不透明球，并排一眼见；④阴影：Opaque 幕投整块方影，
Mask/Blend/A2C 均投**叶形影**（shadow pass 按 alpha 剪），Add/Multiply 面片投实心影
（半透明不挡光的直觉在影子那里不成立——正文提一句）；⑤unlit+Add 照样吃 alpha_mode——
字段文档那句「unlit 忽略 alpha mode」与实测不符（正文按实测写）。

**24-11 琉璃盏**（实跑 ✓，I/T/R/S 键实测）：
```text
小棠：纸灯笼——diffuse_transmission 0.9，灯在它身后。
小棠：瓷灯笼——diffuse_transmission 0，灯在它身后。
小棠：琉璃盏当中坐，I 拨折射率，T 拨厚度，R 拨磨砂，S 拨底片步数。
```

实测：①玻璃球透出身后景物且**上下颠倒**（透镜倒像，ior 1.52+thickness 0.36）；ior 2.42

可见——transmissive pass 在 transparent pass 之前，实证；但纱的**影子**照常透过玻璃可见
（影子画在不透明地板上）；③启动即 steps=0：玻璃变黑琉璃、身后锣彻底不见，只剩环境反光
（GeneratedEnvironmentMapLight 的兜底折射在本影棚里很暗）；**运行时拨到 0 会残留最后一次
抄的底片**（旧画面冻在镜里）——正文按「启动时定妥」措辞；④纸灯笼（diffuse 0.9）整面
洇出暖光 vs 瓷灯笼（0）只有边缘轮廓光——一对照灯全说明白。
